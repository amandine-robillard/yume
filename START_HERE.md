### Étape 1 : Ouvrir Xcode
```bash
# Ouvrez Xcode depuis Applications
```

### Étape 2 : Créer le Projet
```
File → New → Project
iOS → App
Product Name: Yume
Interface: SwiftUI
Language: Swift
✅ Use SwiftData
```

### Étape 3 : Importer les Fichiers
```
Faites glisser tous les dossiers dans Xcode :
- Models/
- Design/
- Services/
```

→ iOS 17.0+ requis dans Deployment Target

→ Vérifiez Target Membership des fichiers
